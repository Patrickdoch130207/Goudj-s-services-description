## About

**Goudj's Services** is a marketplace platform where users can browse, buy and manage products and services. It comes with a fully featured customer-facing storefront and a dedicated admin panel to manage the whole platform.

---

## Key Features

### Customer side

| Feature | Description |
|---|---|
| Sign up & Log in | Create an account and access the platform securely |
| Browse products | Explore all available products and services |
| Product details | View full product info before buying |
| Shopping cart | Add items, update quantities, remove items |
| Checkout | Place orders smoothly |
| Order history | Keep track of past purchases |

### Admin side

| Feature | Description |
|---|---|
| Dashboard | Quick overview of the platform activity |
| Product management | Add, edit and delete products |
| Order management | View and update order statuses |
| User management | Browse and manage registered users |
